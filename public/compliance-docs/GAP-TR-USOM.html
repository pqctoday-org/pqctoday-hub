Siber Güvenlik Başkanlığı; 177 sayılı Cumhurbaşkanlığı Kararnamesi ve 12.03.2025 tarihli, 7545 sayılı Siber Güvenlik Kanunu uyarınca, ülkemizin kritik altyapı sistemlerinin siber dayanıklılığının artırılması ve siber tehditlere karşı dijital güvenliğin tesis edilmesi amacıyla kurulmuştur.

Bu doğrultuda, bugüne kadar www.usom.gov.tr üzerinden sunulan;

  • CVE başvuruları ve siber ihbar süreçleri,
  • Zararlı bağlantı listeleri,
  • Güvenlik bildirimleri ve teknik içerikler,

ilgili mevzuat hükümleri çerçevesinde www.siberguvenlik.gov.tr adresi üzerinden yayımlanmaya devam edecektir.

Zararlı bağlantı listelerinin paylaşım yöntemlerinde güncel standartlara geçiş yapılmaktadır. Mevcut API servislerimiz üzerinden sunulan hizmetlerin sürekliliği önceliğimiz olup, .txt formatındaki dosya paylaşımı 1 Haziran 2026 tarihi itibariyle sona erecektir.

Sistemlerinin güncel kalmasını amaçlayan paydaşlarımızın ve kullanıcılarımızın, veri erişim süreçlerini API servislerimiz üzerinden yürütecek şekilde güncellemeleri tavsiye edilmektedir.

Kamuoyuna saygıyla duyurulur.
